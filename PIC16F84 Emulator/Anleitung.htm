2         Datei öffnen

3         Bedienung der Simulation

3.1       Ablaufkontrolle

Die Simulation eines geöffneten Listings kann über das Fenster Controls gestartet, gestoppt und in Schritten abgearbeitet werden. Die einstellbare Quarzfrequenz mit der der PIC arbeitet kann im Eingabefeld neben den Kontrollknöpfen eingestellt werden. Diese wird in 1000-facher Zeitlupe simuliert. Die maximale zeitliche Auflösung wird bei 4MHz erreicht, die Zykluszeit würde in der Realität 1µs betragen und im Emulator 1ms. Das entspricht einer realen Frequenz von 4kHz. Um die maximale Auflösung zu erreichen, muss der simulierende PC über ausreichend Rechenleistung verfügen.

Mit Doppelklick auf einen Befehl im geöffneten Listing werden Breakpoints umgeschaltet. Ein Breakpoint ist erkennbar am blauen Hintergrund.

 

Die Zeile, die als nächstes abgearbeitet wird, ist orange hinterlegt.

3.2       Output / Input

3.2.1       Fenster Register Map

Der Speicher des Programms wird in der Register Map angezeigt und aktualisiert. Wenn ein Wert aktualisiert wurde, wird dieser kurzzeitig orange hinterlegt. Doppelklick auf einen Wert ermöglicht es, den Wert zu ändern. Als Eingabe ist nur ein Byte in hexadezimaler Schreibweise erlaubt (z.B. "0A") und bestätigt wird die Eingabe mit ENTER.

3.2.2       Fenster I/O Control

Input/Output Funktionalitäten für PORTA und PORTB werden zusätzlich im Fenster I/O Control angezeigt. Mit den Checkboxen können einzelne Bits gesetzt und gelöscht werden. Der Zustand der TRIS-Register wird über den Bits des jeweiligen Ports angezeigt. "i" (Input) steht für eine 1 und "o" (Output) steht für eine 0.

3.2.3       Fenster Special Values

Im Fenster Special Values werden folgende Werte angezeigt:

·         Inhalt des gesamten Stacks (nur Ausgabe)

·         W-Register (veränderbar)

·         Program Counter (veränderbar)

·         Timer 0 (veränderbar)

·         Anzahl der ausgeführten Zyklen (nur Ausgabe)

·         Laufzeit

3.2.4       Fenster Pulse Generator

Im Fenster Pulse Generator kann ein Takt an ausgewählten Bits von PORTA oder PORTB mit einem einstellbaren Intervall generiert werden.

Alle Fenster werden automatisch beim Öffnen eines Listings geöffnet. Über das Menü "Ansicht" können die Fenster separat geöffnet und geschlossen werden.

Die Fenster werden bei folgenden Aktionen automatisch angeordnet:

·         Ein Listing wird geöffnet

·         Die Größe des Emulator-Fensters wird geändert

·         Ein Fenster wird geschlossen / geöffnet
